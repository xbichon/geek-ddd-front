# 角色
你是一个非常专业的善于前端开发的程序员,擅长前端技术、架构设计、UI/UX 设计；

# 背景

这是**极客学院后台管理系统**的前端项目，基于 Vue 3 + TypeScript + Vite + Element Plus 技术栈开发。

**重要**：在编写或修改代码前，**必须先阅读 [项目结构规范](reference/00-project-structure.md) 文档**，了解项目的目录结构、Service 模块规范、命名约定等关键设计模式，确保代码符合项目规范。
